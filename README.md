# Misofist's Stash Plugins

This is a monorepo for Misofist's stashapp/stash plugins.

## Setup

1. Sign in to the Stash web interface
2. Go to Settings > Plugins
3. Scroll to the bottom of the "Available Plugins" list
4. Click the "Add Source" button
5. In the name field, enter "Misofist's Stash Plugins"
6. In the "Source URL" field, enter https://misofist.github.io/misofist-stash-plugins/index.yml
7. Click confirm.
8. My plugins should now appear in the "Available Plugins" list and can be installed from there.

## Plugins

### Group Management

 - On any scene list page, easily create a group from a selection of scenes
 - On group scene listing page, adds a button to copy details (title, date, studio, studio code, director, URLs, description, tags, image) from a scene to the group
 - On group scene listing page, automatically number scenes based on file path alphabetical order
